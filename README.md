Post From Facebook Basic Programming Language public group (https://www.facebook.com/groups/2057165187928233) :
In looking through an old box of stuff from the 1970's, I discovered that I still had a printed copy of a BASIC program I wrote in 1975.  It was a special credits program I did in my last year at university.
I was about to complete my degree in Microbiology at California State University at Long Beach in California but that same year, I'd 'discovered' computers and I'd literally been spending every weekend and evening down in the computer center.
To put my new skills to the test, I'd arraigned with a physics or math professor (can't recall which) to do an extra credits project to calculate the orbits of satellites.   
Well. you can imagine my surprise when I pulled this old program out of the box - 50 years after I'd last seen it.
The program worked and I did get the credits.   It was written on a PDP-11 using the version of BASIC the machine used then.  
Ha!  I still remember when the monitors were upgraded from 110 baud to 300 and then 1200.  Whoo-hoo.
I've photographed the seven pages of the program and placed them onto my Dropbox.  The link to that location is below.
If anyone want to tinker with the code, please, be my guest.
If you do, I'd love to hear the results.   Cheers from Christchurch, New Zealand - where I've been living now since 2009.
Link:  https://www.dropbox.com/.../AHLI9Up0KYnxS3nWo3sH4hE...
